    Permissions

    Tested with PermissionsEx, there are a number of self-explanatory permissions:

    Regeneration

    Allow players to regenerate on death 

    projectrassilon.regen.timelord

    Allow players to force themselves to regenerate 

    projectrassilon.regen.force

    Allow players to block regeneration 

    projectrassilon.regen.block

    Allow players to view their regeneration stats

    This permission is given to all players by default!

    projectrassilon.regen.cmd

    Moderation

    Allow players to view another player's regeneration stats 

    projectrassilon.pr.view

    Allow players to set another player's regeneration count 

    projectrassilon.pr.set

    Allow players to force another player to regenerate 

    projectrassilon.pr.force

    Allow players to block another player's regeneration 

    projectrassilon.pr.block

    Allow players to reload the configuration files 

    projectrassilon.pr.reload
