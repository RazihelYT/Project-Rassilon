    Installation/Setup

    1. Downloading the plugin

    You install ProjectRassilon just like you would any other Bukkit plugin, by dropping it into the Bukkit plugins folder and starting the server. You can download the latest version on BukkitDev.

    The plugin currently requires CraftBukkit1.7.8-R0.1or higher. It is also recommended that you install Vault so you can use regeneration signs.

    2. Configuration

    To generate the configuration files, just start and stop your server. This will also help you verify if you are using the correct CraftBukkit version and have the required dependencies installed.

    After you have stopped your server, navigate to /plugins/ProjectRassilon/ and open the file named config.yml. Edit the file to your liking, then start your server.
